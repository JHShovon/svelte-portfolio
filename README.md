## ğŸš€ Quick Start

### Prerequisites

-   Node.js (version 16 or higher)
-   npm (or yarn)

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/JHShovon/svelte-portfolio.git
    cd svelte-portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start development server:**
    ```bash
    npm run dev
    ```
